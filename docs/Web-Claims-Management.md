# 🛡️ 網頁端領地權限管理

玩家可在網頁端直觀管理個人名下的保護領地，並為好友設定精細的共建權限。

![網頁領地權限管理](./image/dash-5.png)

---

## 🗺️ 1. 領地列表與資訊

在 **「領地管理」** 面板中，會列出您目前擁有的所有領地卡片，包含：
* 領地流水號 ID
* 領地對角座標範圍
* 所屬世界維度 (如 `minecraft:overworld`)

---

## ⚙️ 2. 四大核心權限配置

您可以針對指定領地，為其他玩家（輸入玩家名稱）設定以下四種權限：
1. **Build (建造)**：是否允許放置方塊。
2. **Break (破壞)**：是否允許破壞方塊。
3. **Containers (容器)**：是否允許開啟箱子、熔爐、木桶等儲存容器。
4. **Interact (互動)**：是否允許使用按鈕、拉桿、門、壓力板等紅石互動機關。

您可以隨時在網頁上勾選/取消勾選對應權限，或是一鍵撤銷（移除）該玩家在該領地的所有存取授權。

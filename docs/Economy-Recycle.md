# ⚖️ 自由市場與富豪排行榜

為了促進玩家之間的物資流通、保障自動化量產樂趣，本伺服器採用 **100% 玩家主導的自由市場經濟系統**。

---

## 🚫 1. 系統收購功能關閉

本伺服器**已全面關閉系統回收功能**（即原本的 `/economy` 物資直接換金幣介面）：
* 玩家無法將煤炭、鐵錠、鑽石、石頭或雜物等物資直接拋售給系統來換取金幣。
* 伺服器中沒有「系統無限印鈔」的金幣發放管道，所有金幣均靠交易與簽到流通。

### 💡 玩家獲取金幣的管道：
1. **玩家間相互交易**：透過建立 [箱子商店](Shop-Creation.md) 出售物資給其他玩家。
2. **每日簽到與幸運抽獎**：每日透過遊戲內指令 `/checkin`、`/luckydraw` 或 Discord 連動機器人進行簽到抽獎，可穩定獲得金幣。
3. **商人生涯營收**：經營商店，提領營業金。

---

## 🏆 2. 全服富豪排行榜 (`/eco top` 或 `/economy top`)

玩家可以在遊戲內輸入 **`/eco top`**（或 **/economy top**）開啟伺服器富豪排行榜選單，查閱排行：
* **財富排行**：系統會實時彙整全伺服器所有玩家的帳戶餘額（包含離線玩家），並在選單中依序顯示出金幣排名前 **10 名** 的富豪玩家。
* **跨平台資料同步**：該財富排名會即時同步至 Discord 機器人及網頁端儀表板，讓您在遊戲外也能隨時掌握伺服器的財富榜！

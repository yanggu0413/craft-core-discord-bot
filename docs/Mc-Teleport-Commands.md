# 🧭 遊戲內傳送與地標指南

為了讓玩家在廣闊的世界中自由穿梭，Craft-Core 伺服器提供了完整的 **傳送請求 (TPA)**、**個人家園 (Homes)** 以及 **公共地標與設施審核系統 (Warps & Audit)**。所有傳送指令皆已支援**智慧型 Tab 自動補全與選擇器下拉選單**。

---

## 👥 1. 玩家間傳送指令 (Teleport Requests / TPA)

若想與其他線上玩家互傳，可以使用以下指令（已內建線上玩家選單自動補全）：

| 指令 | 說明 | 參數說明 |
| :--- | :--- | :--- |
| **`/tpa <玩家名稱>`** | 向指定線上玩家發送傳送請求，希望傳送到對方身邊 | 支援 Tab 自動選擇線上玩家 |
| **`/tpahere <玩家名稱>`** | 向指定線上玩家發送傳送請求，希望對方傳送到您身邊 | 支援 Tab 自動選擇線上玩家 |
| **`/tpaccept <玩家名稱>`** | 接受來自該玩家發送給您的傳送請求 | 可直接點擊聊天欄按鈕或 Tab 補全 |
| **`/tpdeny <玩家名稱>`** | 拒絕來自該玩家發送給您的傳送請求 | 可直接點擊聊天欄按鈕或 Tab 補全 |
| **`/tpa cancel`** | 取消您當前已發送給其他玩家的傳送請求 | 取消後可重新發送 |

---

## 🏠 2. 個人家園傳送指令 (Player Homes)

玩家可以在任何安全的角落建立個人專屬的家園傳送點：

| 指令 | 說明 | 備註 |
| :--- | :--- | :--- |
| **`/sethome <家名稱>`** | 在您當前站立的位置設定名為 `<家名稱>` 的家 | 若未輸入名稱預設為 `home` |
| **`/home <家名稱>`** | 瞬間傳送至名為 `<家名稱>` 的家 | 支援下拉選單列出您已建立的家 |
| **`/delhome <家名稱>`** | 刪除您已設定的名為 `<家名稱>` 的家 | 支援選單自動補全 |
| **`/home list`** | 列出您目前擁有的所有家名稱與座標 | 也可在網頁儀表板管理 |

---

## 🏁 3. 公共地標傳送指令 (Warps)

公共地標為官方或玩家審核通過的公共區域傳送點：

| 指令 | 說明 | 權限/備註 |
| :--- | :--- | :--- |
| **`/warp <地標名稱>`** | 瞬間傳送至指定公共地標（如 `/warp spawn`） | 所有人皆可使用，支援選單自動補全 |
| **`/warp list`** | 列出伺服器中所有的公共地標與座標 | 所有人皆可使用 |
| **`/setwarp <地標名稱>`** | 在當前位置建立新的公共地標 | 僅限 OP 管理員使用 |
| **`/delwarp <地標名稱>`** | 刪除指定的地標 | 僅限 OP 管理員使用 |

---

## 📍 4. 公共設施與 Warp 投稿審核系統

為了鼓勵玩家建造刷鐵機、公共農場、紅石設施與玩家小鎮，伺服器提供**公共設施審核與 Warp 自動建立流程**：

### 🤖 Discord 頻道審核 (`1524354515661492344`)
1. 前往 Discord **`1524354515661492344`** 頻道。
2. 檢視現有公共傳送點列表 Embed，並點擊 **`📝 提交公共設施 / Warp 審核`** 按鈕。
3. 在彈出的 Modal 表單中填寫：
   - **設施名稱**（如 `刷鐵機` / `公共農場`）
   - **設施功能說明**（如 `免費提供鐵錠與紅石物資`）
   - **設施座標 (X Y Z)**（如 `150 64 -200`）
   - **所在世界**（預設為 `overworld` 主世界）
4. 提交後自動送交管理員頻道審核，管理員點擊 `✅ 同意通過` 時，系統會**自動在遊戲內設立 `/warp <名稱>` 並同步更新 Discord 與網頁清單**！

### 🖥️ Web Dashboard 傳送與審核管理 (`/teleports`)
- 前往網頁儀表板的 **「傳送點管理」** 分頁：
  - **我的家園**：即時查看與遠端刪除個人 Home 點。
  - **公共地標**：檢視所有公共 Warp 點與座標。
  - **設施審核**：線上提交設施審核，管理員可於此直接進行 `✅ 核准` 與 `❌ 駁回`。

---

## 🏪 5. 商店傳送機制澄清

* 普通玩家在伺服器中**沒有直接輸入 `/tp <X> <Y> <Z>` 任意座標傳送的權限**。
* 一般玩家若想傳送至某個玩家箱子商店，請在遊戲內輸入 **`/shop`** 開啟選單，點選商品後點擊 **`[傳送到商店]`** 進行安全傳送。

# 🧭 遊戲內傳送與地標指南

為了讓玩家在廣闊的世界中更方便穿梭，本伺服器提供了完整的傳送請求 (TPA)、個人家園 (Homes) 以及公共地標 (Warps) 系統。

---

## 👥 1. 玩家間傳送 (Teleport Requests)

若想與其他玩家互傳，可以使用以下指令：
* **`/tpa <玩家名稱>`**：發送請求，希望能傳送到該玩家的身邊。
* **`/tpahere <玩家名稱>`**：發送請求，希望將該玩家傳送到您當前的位置。
* **`/tpaccept <玩家名稱>`**：接受該玩家發送給您的傳送請求。
* **`/tpdeny <玩家名稱>`**：拒絕該玩家發送給您的傳送請求。

---

## 🏠 2. 個人家園 (Player Homes)

設定您個人的傳送點（不限數量），方便隨時傳送回家：
* **`/home set <家名稱>`**：在您當前站立的位置設定一個名為 `<家名稱>` 的家。
* **`/home tp <家名稱>`**：瞬間傳送至名為 `<家名稱>` 的家。
* **`/home delete <家名稱>`**：刪除您已設定的名為 `<家名稱>` 的家。
* **`/home list`**：列出您目前擁有的所有家名稱。

---

## 🏁 3. 地標傳送 (Warps)

公共地標是由管理員設定的公共區域傳送點，所有玩家皆可傳送：
* **`/warp tp <地標名稱>`**：瞬間傳送至名為 `<地標名稱>` 的公共地標。
* **`/warp list`**：列出伺服器中所有的公共地標。
* *管理員專用指令*：
  - `/warp set <地標名稱>`：在當前位置建立公共地標。
  - `/warp delete <地標名稱>`：刪除指定的地標。

---

## 🏪 4. 商店傳送與權限限制

* **一般玩家傳送方式**：
  * 普通玩家在伺服器中**沒有 `/tp` 的執行權限**。
  * 網頁端商店瀏覽器複製的 `/tp <X> <Y> <Z>` 指令**僅限 OP 管理員使用**。
  * 一般玩家若想傳送至某個玩家商店，請在遊戲內輸入 **`/shop`** 開啟選單，點選商品後點選中間的 **`[傳送到商店]`（GUI 中的 Slot 10）** 進行安全傳送。
* **指令功能澄清**：
  * 指令 `/shop control` 是店主與管理員進行遠端操作的後台命令（例如補貨、切換模式等），不具備直接提供買家傳送的功能。

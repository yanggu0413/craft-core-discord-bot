# 🛠️ 管理員指令與商店管理

本章節列出伺服器管理員（OP）專用的維護與管理指令，以及店主遠端管理的相關權限。

---

## 🏪 1. 管理員無限商店設定

管理員可以建立免庫存、免資金的無限商店（系統商店）：
1. 按照常規方式建立商店，或對著現有商店告示牌點選 **【右鍵】** 開啟店主管理面板。
2. 只有管理員（OP）能看見 **「無限模式」** 狀態，點選右側的 **`§d[切換無限]`** 按鈕進行切換啟用。
3. 啟用後，商店庫存會顯示為 `無限`，玩家可以無限制地進行交易。
4. **遠端清空箱子**：對著無限商店，點擊管理面板中的 **`[清空箱子]`** 按鈕，即可快速清空實體箱子中該商店類型的物品。此指令僅限 OP 與無限商店店主本人執行。

---

## 📦 2. 店主遠端補貨機制

伺服器對**所有商店店主**開放了遠端補貨功能：
1. 店主在管理面板（Chat）點選 **`[遠端補貨]`** 按鈕。
2. 系統會自動加載對應的區塊，為店主開啟一個 **27 格的補貨箱子 UI 介面**。
3. **補貨過濾與安全性防吞機制**：
   * 補貨 UI 中**只能放置與該商店販售目標相同的物品**。
   * 當店主關閉補貨介面時，系統會將放進去的相符物資真實移入地圖中的實體箱子，並更新商店庫存。
   * 任何不相符的物品，或實體箱子空間已滿而裝不下的多餘物資，將在關閉時**安全退回到您的玩家背包中**（若背包已滿則會掉落於您身邊），保證物資絕對安全不遺失！

---

## ⚙️ 3. 全局管理與糾紛處理指令

* **`/shop list`**：列出全服所有商店的詳細資訊與擁有者。
* **`/shop reload`**：重新載入商店資料檔與設定檔。
* **強拆商店**：管理員可以直接破壞任何玩家的商店告示牌，系統會強制註銷並清理該位置的商店數據。

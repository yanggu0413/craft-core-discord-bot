# 🛠️ 管理員指令與商店管理

本章節列出伺服器管理員（OP）專用的維護與管理指令，用於監控全服商店與解決糾紛。

---

## 🏪 1. 管理員無限商店設定

管理員可以建立免庫存、免資金的無限商店：
1. 按照常規方式建立商店，或對著現有商店告示牌點選 **【右鍵】** 開啟管理面板。
2. 點選 **「無限模式 (Infinite Mode)」** 旁邊的 `[修改]` 將其啟用。
3. 啟用後，商店庫存會顯示為 `無限`，玩家可以無限制地進行交易。
4. **遠端補貨與清空**：無限商店的管理面板會額外解鎖 `遠端補貨` 與 `清空商品` 的 `[執行]` 按鈕，點擊可快速操作。

---

## ⚙️ 2. 全局管理與糾紛處理指令

* **`/shop list`**：列出全服所有商店的詳細資訊與擁有者。
* **`/shop reload`**：重新載入商店資料檔與設定檔。
* **強拆商店**：管理員可以直接破壞任何玩家的商店告示牌，系統會強制註銷並清理該位置的商店數據。

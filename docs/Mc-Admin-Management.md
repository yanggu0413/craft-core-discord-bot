# 🛠️ 管理員指令與管理主控台指南

本章節完整列出 Craft-Core 伺服器管理員（OP）專用的遊戲內指令、Discord 管理員控制面板，以及 Web Dashboard 管理主控台的相關操作說明。

---

## ⚙️ 1. 遊戲內管理員（OP）專用指令清單

| 指令 | 說明 | 參數範例 | 備註 |
| :--- | :--- | :--- | :--- |
| **`/addmoney <玩家> <金額>`** | 增加指定玩家的遊戲金幣餘額 | `/addmoney Alex 5000` | 無需目標在線，金幣即時到帳 |
| **`/removemoney <玩家> <金額>`** | 扣除指定玩家的遊戲金幣餘額 | `/removemoney Alex 1000` | 最多扣至 $0，不會為負數 |
| **`/setwarp <地標名稱>`** | 在管理員當前位置建立公共地標 | `/setwarp spawn` | 全服玩家皆可透過 `/warp <名稱>` 傳送 |
| **`/delwarp <地標名稱>`** | 刪除指定的公共地標 | `/delwarp spawn` | 刪除後自動同步網頁與 Discord 列表 |
| **`/invsee <玩家>`** | 遠端開啟並即時查看/編輯指定線上玩家的背包與裝備箱 | `/invsee Alex` | 由伺服器整合包獨立背包模組提供 |
| **`/ban <玩家> [原因]`** | 封鎖指定玩家並禁止登入 | `/ban BadUser 破壞地形` | 自動發送廣播與紀錄 |
| **`/kick <玩家> [原因]`** | 強制踢出指定線上玩家 | `/kick AFKUser 佔用伺服器` | 踢出後可重新連線 |
| **`/fp <名稱> <動作>`** | 管理員控制與清理假人 | `/fp fp_alex kill` | 管理員擁有最高權限可強制下線任何假人 |
| **`/shop list`** | 列出全服所有註冊商店 | `/shop list` | 包含商店位置、擁有者與物品 |
| **`/shop reload`** | 重新載入商店資料與設定檔 | `/shop reload` | 當手動修改 JSON 設定檔時使用 |

## 🏪 2. 管理員無限商店（系統商店）

管理員可建立免庫存、免資金的官方系統商店：
1. 按照常規方式放置箱子與告示牌建立商店，或對著現有商店告示牌點選 **【右鍵】** 開啟店主管理面板。
2. 僅管理員（OP）能看見 **「無限模式」** 狀態，點選右側的 **`§d[切換無限]`** 按鈕進行切換。
3. 啟用後，商店庫存會顯示為 `無限`，玩家可以無限制地進行買賣。
4. **遠端清空箱子**：對著無限商店，點擊管理面板中的 **`[清空箱子]`** 按鈕，即可快速清空實體箱子中該商店類型的物品。

## 🤖 3. Discord 管理員控制面板 (頻道 `1524977578362933419`)

專為管理員設計的 Discord 遠端維護控制台，提供以下核心功能按鈕：
* **🚫 封鎖玩家 (admin_ban)**：彈出 Modal 輸入玩家 ID 與理由，自動發送 `/ban` 命令。
* **🥾 踢出玩家 (admin_kick)**：彈出 Modal 輸入玩家 ID 與理由，自動發送 `/kick` 命令。
* **🤝 聯名獎勵 (admin_co_brand)**：輸入玩家名稱或 Discord ID，自動獎勵 **$5,000 遊戲幣 + 6 把抽獎鑰匙**！
* **🔍 查詢玩家 (admin_search_player)**：查詢指定玩家的餘額、線上/離線狀態、座標、TPS、簽到天數與背包物品。
* **📢 發布全服公告 (admin_draft_announcement)**：填寫公告主題與內文，進行預覽、發布或廢棄，發布時於遊戲內發送全服彩色廣播。
* **📍 公共設施 Warp 審核審查**：接收來自玩家提交的設施審核卡片，點擊 `✅ 同意通過並設立 Warp` 即可自動設立遊戲內地標並更新頻道 Embed！

## 🖥️ 4. Web Dashboard 管理員主控台 (`/admin`)

網頁端儀表板為擁有管理員身分（`isAdmin`）的帳號提供專屬控制台：
1. **玩家管理面板**：可快速進行玩家 Ban/Kick 操作，並發送 $5,000 + 6 鑰匙聯名獎勵。
2. **玩家背包與個人資產檢視**：輸入玩家名稱即可調閱其金幣餘額、座標、TPS、簽到數據與實體背包物品清單。
3. **🎪 伺服器活動發布與管理 (`/api/admin/events`)**：
   - 建立全新的限時節慶、採礦雙倍、抽獎大賽活動。
   - 設定活動標題、詳細說明、起訖時間、豐厚獎勵，並切換狀態 (`active` 🟢 / `paused` ⏸️ / `completed` ⚪)。
4. **📍 公共設施 Warp 審核列表 (`/api/admin/warp-submissions`)**：
   - 線上檢視所有玩家提交的公共設施（設施名稱、功能說明、座標、維度）。
   - 提供一鍵 `✅ 同意通過`（自動建立 `/warp` 點）與 `❌ 駁回` 按鈕。

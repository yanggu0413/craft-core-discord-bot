# 🎁 網頁端每日簽到與幸運抽獎

伺服器每天都為玩家提供豐富的福利！您可以在網頁端一鍵完成簽到與抽獎。

![網頁簽到與抽獎福利](./image/dash-7.png)

---

## 📅 1. 每日連續簽到

* 進入網頁版 **「每日福利」** 頁面。
* 首頁會顯示您當前的 **連續簽到天數**、**累計簽到次數** 以及上次簽到時間。
* 每天可點擊簽到按鈕完成簽到，獲取金幣獎勵與 **幸運抽獎鑰匙 🔑**。

---

## 🎟️ 2. 幸運抽獎系統

* 簽到獲得的鑰匙可用於幸運抽獎。
* 網頁端會顯示您目前擁有的鑰匙數量。
* 點擊抽獎按鈕，即可消耗鑰匙進行物資抽獎，抽中的物資會直接寄送到您的「快遞包裹」或「遊戲內郵箱」中！

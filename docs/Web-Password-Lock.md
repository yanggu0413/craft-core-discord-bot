# 🔒 網頁端密碼箱白名單管理

安全密碼箱提供了強大的個人財產保護。除了在遊戲內設定密碼，您可以在網頁端管理授權白名單。

---

## 📁 1. 密碼箱名冊

在 **「密碼箱管理」** 分頁中：
* 系統會列出您在遊戲中放置並上鎖的全部密碼箱位置。
* 每張卡片會顯示該密碼箱的確切世界座標。

---

## 👥 2. 授權與白名單管理

您是密碼箱的唯一擁有者，您可以遠端進行以下操作：
* **新增白名單成員**：在卡片中輸入好友的 Minecraft ID，將其加入開啟白名單。加入後，該玩家即使不輸入密碼，也可以直接打開該箱子！
* **撤銷授權**：點擊已授權玩家名字旁邊的 **`[ × ]`** 鈕，即可立刻撤銷該玩家的開啟權限。
* **遠端變更密碼**：點擊 **「修改密碼」**，輸入新密碼即可即時生效。
* **註銷密碼鎖**：點擊紅色的 **「註銷密碼鎖」**，徹底解除箱子的密碼鎖定防護，使其變回普通箱子。

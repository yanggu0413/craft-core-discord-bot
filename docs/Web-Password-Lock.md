# 🔒 密碼安全箱設定與網頁端白名單管理

安全密碼鎖系統可讓您為個人的貴重箱子（或木桶）提供雙重保護。即使該箱子沒有位於您的領地內，只要上鎖後，其餘人一概無法開啟或破壞。

---

## 🎮 1. 遊戲內設定密碼鎖

### 【步驟一】發起上鎖
1. 站在您想要上鎖的箱子（或木桶）前。
2. 雙眼看著該箱子（距離必須在 4.5 格內）。
3. 在聊天欄輸入指令：**`/padlock`**。
4. 系統會攔截您的下一次聊天發言，並在聊天欄顯示提示：`§b[Craft-Core] §f請輸入密碼以設定密碼鎖：`。

### 【步驟二】輸入密碼
* 直接在聊天欄打出您想要設定的密碼並按 Enter 發送（該訊息會被系統攔截，不會被其他玩家看見）。
* 設定完成後會播放 Chest Locked 聲效並顯示：`§b[Craft-Core] §a密碼鎖設定成功！`。

---

## 👥 2. 共享與白名單授權

上鎖後的箱子，預設僅有**您本人**與**管理員(OP)**可以開啟。若要讓好友免密碼直接開啟：

### 遊戲內授權指令：
1. 站在該密碼箱前，看著它（4.5 格內）。
2. 輸入指令：**`/padlock grant <好友名稱>`**。
3. 系統即會成功將其加入授權名冊中。

### 網頁端白名單管理：
登入網頁儀表板後，點選 **「密碼箱管理」** 分頁，即可遠端對名下的所有安全箱進行管理：
* **新增授權白名單**：直接在安全箱卡片中輸入好友的 Minecraft ID，將其加入白名單。
* **撤銷授權**：點選好友名字旁邊的 **`[ × ]`** 即可立刻取消其開啟權限。
* **遠端修改密碼**：點選「修改密碼」即可直接於網頁變更安全箱密碼。
* **遠端註銷密碼鎖**：點選「註銷密碼鎖」可將密碼箱回復成一般箱子。

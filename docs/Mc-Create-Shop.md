# 🔨 如何建立遊戲內箱子商店

只要幾個簡單的步驟，您就可以在您的領地中建立一個自動交易商店。

---

## 📝 1. 告示牌書寫格式

在箱子（或木桶）的**正面或上方**放置一個告示牌，並依序輸入以下四行內容：

```text
[第一行] [Shop]
[第二行] 交易數量 (如: 64)
[第三行] 出售價:收購價 (如: B 100:10 S)
[第四行] 物品 ID 或手持 [this] 代替
```

### ✍️ 參數詳細說明：
* **第一行**：必須輸入 **`[Shop]`**（大小寫均可，建立成功後會自動變為黃色加粗 `§e§l[Shop]`）。
* **第二行**：每次交易的物品數量（必須是正整數，例如 `1` 或 `64`）。
* **第三行**：
  - **`B` 代表買入 (Buy)**：買家跟您買的價格。例如 `B 100`。
  - **`S` 代表收購 (Sell)**：您向買家回收的價格。例如 `S 10`。
  - **雙向設定**：例如 `B 100:10 S`（代表買家花 $100 買，或把商品以 $10 賣還給您）。
* **第四行**：
  - 手持您要販售的物品，在第四行輸入 **`[this]`**（或英文 `this`）。
  - 系統會自動辨識您手上的物品，並自動填入正確的物品名稱（如 `DIAMOND`），避免輸入複雜的 ID！

# 📖 CraftCoreShop Wiki 首頁

歡迎來到 **CraftCoreShop** 官方使用與設定指南！

本系統是一套專為 Minecraft 伺服器打造的**純服務端商店與虛擬經濟系統**。玩家無需安裝任何客端模組即可享受精美的遊戲內介面與安全快捷的交易體驗。

---

### 1. 📦 商店系統詳細指南

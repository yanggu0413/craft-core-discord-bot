# 🎪 伺服器活動系統 (Server Events System)

Craft-Core 伺服器提供全新的 **伺服器活動系統**，包含限時節慶、雙倍經驗/採礦競賽、集字抽獎與全服狂歡節。玩家可隨時在遊戲內或網頁端關注最新的活動動態並領取豐厚獎勵！

---

## 🎮 1. 遊戲內活動指令與自動提醒

* **`/events`**：在遊戲內聊天欄輸入 `/events`，系統將即時發送當前所有正在進行中的熱門活動標題、詳細說明、時間與豐厚獎勵！
* **登入自動提醒 (Join Broadcast)**：每當您登入伺服器時，系統會在連線成功 2 秒後自動發送彩色限時活動通知，確保您不漏接任何福利訊息。

---

## 🖥️ 2. Web Dashboard 活動專區 (`/events`)

前往網頁儀表板的 **「🎪 伺服器活動」** 分頁：
1. **熱門活動看板**：以卡片形式精美展示全服活動，包含活動狀態標籤：
   - 🟢 **限時進行中 (Active)**：正開放玩家參與中。
   - ⏸️ **暫停 (Paused)**：暫時休止。
   - ⚪ **已結束 (Completed)**：歷屆存檔紀錄。
2. **首頁熱門活動 Alert Banner**：登入儀表板首頁 (`HomeView`) 頂部會自動浮現動態亮眼的「熱門限時活動 Banner」，可一鍵點擊跳轉至活動專區。
3. **活動詳情與獎勵**：清楚載入活動規則、開辦時間、結束時間與獎勵項目（如高額遊戲金幣、抽獎鑰匙與稀有道具）。

---

## 🛠️ 3. 管理員發布與維護專區 (Admin Event Management)

管理員在網頁儀表板 (`/events` 或 `/admin`) 可進行以下管理：
- **發布新活動**：點選「發布新伺服器活動」按鈕，填寫活動標題、詳細內容、起訖時間與獎勵說明。
- **即時狀態切換**：可隨時一鍵切換活動狀態（啟用/暫停/結束），系統將同步即時更新遊戲內與網頁端。
- **編輯與刪除**：隨時修改活動條款或刪除過期活動。

# 🤖 Discord 機器人連動指南

本系統配備了強大的 **Discord 雙向連動機器人**。玩家與管理員可直接在 Discord 頻道中點選按鈕、使用選單或斜線指令，與遊戲伺服器進行即時的資料互通與管理。

---

## 1. 🔑 帳號雙向綁定

為了使用機器人的各項個人化功能（如餘額查詢、離線信箱與簽到），您必須先將您的 Discord 帳號與 Minecraft 遊戲角色進行綁定：
1. **獲取驗證碼**：在 Minecraft 遊戲中輸入 `/discord`（或 `/discord bind`），系統會隨機生成一組 **6 位數字驗證碼**（例如：`654321`）。
2. **私訊機器人**：複製該驗證碼，並在 Discord 中私訊本伺服器機器人，直接發送該 6 位數字。
3. **完成綁定**：機器人驗證成功後會回傳綁定成功訊息，並將您的 Discord 帳號與 Minecraft 角色（含 UUID）進行雙向鎖定。
   - *註：若要解除綁定，可在 Discord 輸入 `/unbind` 或 `/解除綁定` 指令。*

---

## 2. 🎮 Key Panel (密鑰控制面板)

密鑰控制面板提供玩家每日福利與提醒訂閱功能：
*   **📅 每日簽到 (Check-in)**：
    - 點選按鈕即可完成當日簽到，系統會隨機發放原版生存道具作為獎勵。
    - **簽到獎金加碼**：不論抽中何種道具，每次簽到皆會**額外加贈 $150 虛擬金幣**！
    - **連續簽到天數 (Streak)**：系統會累計您的連續簽到天數，連續簽到天數越高，獲得高級稀有物資的機率就越大。
*   **🔔 訂閱提醒 (Reminder Subscription)**：
    - 點選「訂閱簽到提醒」按鈕，每當每日簽到在午夜重置後，機器人會主動向您發送 Discord 私訊提醒，避免您的簽到中斷。
*   **🎰 幸運抽獎 (Lucky Draw)**：
    - 點選「幸運抽獎」按鈕可消耗您的遊戲積分進行抽獎。
    - *限制：玩家必須處於遊戲線上狀態（Online）才能進行抽獎。*
    - 每次抽獎皆會**額外保證加碼 $150 遊戲幣**！獎勵會直接在遊戲內以標題（Title）與音效即時呈現。

---

## 3. 🏪 Interaction Panel (互動控制面板)

互動控制面板讓玩家可隨時監控個人資產與發送快遞：
*   **💰 查詢金幣餘額**：
    - 點選按鈕，機器人會即時透過 WebSocket 向遊戲伺服器查詢您當前角色的餘額，並以私密訊息回傳：`您目前的金幣餘額為：$XXXX 元`。
*   **📊 我的商店數據**：
    - 點選按鈕即可列出您目前名下所有箱子商店的詳細數據，包括各商店座標、販售商品、定價、庫存及累積未提領營收。
*   **📦 離線快遞信箱 (Offline Mailbox)**：
    - 當您想轉帳或郵寄金幣給其他玩家，但對方處於離線狀態時，您可以在 Discord 點選「離線快遞」，並於彈出的選單中選擇收件人與輸入金額。
    - 系統會暫存該筆款項。當收件人下次登入遊戲時，伺服器會自動偵測並在聊天欄顯示：`📦 [快遞] 收到來自 XXX 的快遞金幣：$XXX 元！`，並自動將款項撥入其帳戶。

---

## 🛠️ 4. Admin Panel (管理員控制面板)

專為伺服器管理員提供的遠端維護面板（僅限擁有管理員權限之 Discord 角色組點擊）：
*   **🚫 封鎖玩家 (Ban)** / **🥾 踢出玩家 (Kick)**：
    - 點擊按鈕會彈出輸入框，輸入遊戲 ID 與理由，即可跨平台將違規玩家踢出伺服器或加入黑名單。
*   **📢 發布公告 (Announcements)**：
    - 點選按鈕後可輸入公告標題與內容。提交後，遊戲伺服器會以最酷炫的 `tellraw` 彩色全服公告形式即時推播給所有線上玩家。

---

## 📈 5. Economy Panel (經濟與日誌面板)

*   **🏆 全服富豪排行榜 (Leaderboard)**：
    - 點選按鈕，機器人會拉取遊戲內金幣餘額前 10 名的明星玩家，並在 Discord 頻道中渲染出精美的財富排行榜。
*   **📜 交易即時推播頻道 (Real-time Transaction Logs)**：
    - 只要遊戲內有任何箱子商店成交，伺服器就會即時將交易詳情（買家、賣家、商品、單價、交易數量與印花稅）包裝為 JSON 傳送給 Discord。
    - 機器人會將其格式化並發送至指定的交易日誌頻道，讓玩家在 Discord 就能掌握第一手的物價波動！

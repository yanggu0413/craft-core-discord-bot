# 🛠️ 管理員與指令手冊

本章節面向伺服器管理員（擁有 Level 4/OP 權限的使用者或控制台），介紹如何管理與調節伺服器經濟。

---

## 1. ⚙️ 管理員經濟控制指令

管理員可以手動調整或發放金幣（例如作為每日活動獎勵或綁定獎勵）：
*   **增加玩家餘額**：
    `/addmoney <玩家ID> <金額>`
    *   *範例*：`/addmoney im_little_rory 5000`
    *   *作用*：為指定玩家增加 5000 金幣。
*   **扣除玩家餘額**：
    `/removemoney <玩家ID> <金額>`
    *   *範例*：`/removemoney im_little_rory 1000`
    *   *作用*：為指定玩家扣除 1000 金幣。

---

## 2. 🏛️ 建立系統無限商店 (Admin Shop)
管理員可以在伺服器重生點或公共市場建立「系統無限商店」：
1. 進入建立商店模式（手持物品左鍵點選箱子）。
2. 在輸入出售價格或收購價格時，完成商店建立。
3. 作為 OP，您可以使用專屬指令將該商店的擁有者轉移為系統：
   `/shop setowner <座標> Server`
4. 轉移為 `Server` 後：
   - 該商店的商品庫存會變為 **無限（Infinite）**。
   - 該商店的收購金幣會變為 **無限**。
   - 玩家向其購買的金額會被系統直接銷毀；玩家向其出售的金額由系統直接發放。

---

## 3. 🚨 違規商店強制註銷與維護
當發現玩家違規擺放商店或有不當命名時：
*   **強制破壞**：管理員可以直接使用工具（如創造模式下）強行挖掉該商店的箱子或告示牌。系統會自動攔截破壞事件，安全地將該商店從資料庫註銷，並完整清理箱子正上方的縮小懸浮展示物，防止實體殘留。
*   **遠端註銷**：管理員在 `/shop` 選單中點選任何玩家的商店，除了傳送按鈕外，會顯示專屬的管理員紅色按鈕 `[ 強制刪除商店 ]`，點擊即可完成遠端註銷。

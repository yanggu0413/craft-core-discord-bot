# 🛠️ 管理員與維護指南

本章節介紹擁有管理員權限的使用者或控制台，如何調控伺服器經濟、建立系統無限商店，以及對違規商店進行強制維護與清理。

---

## 1. ⚙️ 管理員經濟調控指令

管理員可以手動調整玩家的帳戶金幣：
* **發放金幣給玩家**：
  `/addmoney <玩家ID> <金幣金額>`
  * *範例*：`/addmoney Yanggu 5000`
  * *效果*：為指定玩家的虛擬帳戶增加 5000 金幣。
* **扣除玩家金幣**：
  `/removemoney <玩家ID> <金幣金額>`
  * *範例*：`/removemoney Yanggu 1000`
  * *效果*：扣除指定玩家虛擬帳戶內的 1000 金幣。

---

## 2. 🏛️ 建立「系統無限商店」（Server Shop）

管理員可建立無限買賣的系統商店：
1. 依照一般流程建立商店（手持物品左鍵點擊箱子並完成定價）。
2. 身為管理員，您可以使用右鍵點擊該商店的告示牌，開啟管理面板。
3. 點選 **「無限模式：停用」** 旁邊的 **`[修改]`** 按鈕，將其切換為 **`啟用`**。
4. **系統商店的行為**：
   - 該商店的商品庫存會變為 **無限**，買家可無限量購買。
   - 該商店的收購金幣會變為 **無限**，玩家可無限量賣入以換取金幣。
   - 玩家向系統商店購買商品所支付的金幣會被系統**直接銷毀**，移出市場流通；玩家賣出商品給系統商店獲得的金幣則由系統直接發放。
   - 管理員可使用管理面板中的 `[遠端補貨]` 與 `[清空商品]` 功能來一鍵填滿或排空箱子。

---

## 🚨 3. 違規商店的強制清理與維護

當發現玩家將商店放置於違規區域時，管理員可以進行強制清理：
* **物理強制破壞**：管理員可直接在創造模式下破壞商店箱子或告示牌。系統會立即捕捉此事件，並在系統記錄中安全註銷該商店，同時**清除**箱子正上方的懸浮物品模型，防止殘留。
* **遠端強制註銷**：管理員在 **/shop** 選單中點選任何玩家名下的商店時，選單中除了傳送按鈕外，會多出一個紅色的 **`[ 強制註銷商店 ]`** 按鈕，點擊即可完成遠端清除。

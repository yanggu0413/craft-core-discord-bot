# ❌ 如何刪除與註銷商店

當您不再需要某個商店，或是想要更改販售的商品時，可以將其刪除並註銷。

---

## 🗑️ 1. 方法一：直接敲除告示牌

* 最直觀的方法：直接使用工具（如斧頭）或徒手**敲掉商店箱子上的告示牌**。
* 告示牌破碎的同時，系統會發送通知確認商店已註銷，並會自動清除箱子上方對應的商品懸浮展示模型。

---

## 🖱️ 2. 方法二：透過店主控制面板刪除

1. 店主對著自己商店的 **告示牌點選【右鍵】**。
2. 聊天欄會開啟店主專屬控制面板。
3. 點選最後一行的紅字 **`[刪除商店]`**。
4. 系統會即時移除該商店的註銷狀態，告示牌的文字會自動被清空，方便您重新書寫。

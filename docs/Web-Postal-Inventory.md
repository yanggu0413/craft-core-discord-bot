# 📬 遠端郵局快遞與個人背包

本伺服器提供先進的跨維度離線快遞物流，讓玩家能在網頁上將金幣與物品發送給其他玩家。

---

## 🎒 1. 個人遊戲背包實時預覽 (36格)

* **狀態限制**：當您在遊戲內處於 **線上** 狀態時，此面板會透過 WebSocket 即時同步，在網頁端完整渲染您當前的 36 格背包畫面。
* **發送快遞包裹**：
  1. 直接點選網頁背包中任何一格您想送出的物品。
  2. 在右側欄位中輸入 **「收件人名稱」** 與 **「物品數量」**。
  3. 點擊確認送出，該物品會立即從您遊戲內的背包中消失，並轉入物流系統，寄送給對方。

---

## 💸 2. 金幣電子匯款

* 支援線上與離線匯款。
* 在匯款面板中輸入對方的 Minecraft ID 以及金幣金額，即可直接從您的個人金幣帳戶扣除餘額，以快遞包裹形式發送給對方。

---

## 📦 3. 收件狀態追蹤

所有發送與接收的快遞皆可在網頁端追蹤狀態：
* **`已送達 (Delivered)`**：收件人已上線接收快遞物品。
* **`離線暫存 (Pending)`**：收件人目前離線，物品已安全儲存於伺服器雲端快遞庫中，待收件人下次登入遊戲時會自動收到彈出通知並派發。
